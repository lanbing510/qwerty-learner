# 自定义词典功能使用指南

## 功能概述

自定义词典功能允许用户创建属于自己的英语单词词典，系统会自动从已有的词典中查找单词的释义和发音信息。

## 访问入口

1. 进入词典选择页面（点击首页的"词典"按钮）
2. 点击右上角的 **"创建自定义词典"** 按钮
3. 或者在"我的自定义词典"区域，悬停在已有词典卡片上，点击出现的 **编辑图标**（蓝色铅笔图标）

---

## 一、创建新词典

### 步骤 1：填写基本信息

- **词典名称**（必填）：例如"我的三年级单词本"
- **词典描述**（可选）：简短描述

### 步骤 2：输入单词列表

在单词列表文本框中输入英文单词，支持以下格式：
- 每行一个单词
- 用逗号分隔
- 用空格分隔

示例：
```
apple, banana, orange
cat, dog, bird
flower, tree, sun
```

### 步骤 3：解析单词

点击 **"解析单词"** 按钮，系统会：
1. 自动从现有词典中查找每个单词的释义和音标
2. 显示查找进度
3. 所有单词将作为**一个章节**保存

### 步骤 4：预览和保存

- 预览区域会显示所有章节和单词
- 确认无误后点击 **"保存词典"**
- 保存后在词典列表的"我的自定义词典"区域即可看到

---

## 二、编辑已有词典

在词典列表中，悬停在自定义词典卡片上，点击出现的 **蓝色编辑图标** 进入编辑页面。

### 可编辑的内容

1. **词典基本信息**
   - 修改词典名称
   - 修改词典描述
   - 点击"保存全部"按钮保存修改

2. **编辑单词**
   - 点击单词旁边的编辑图标（蓝色铅笔）
   - 可以修改：单词拼写、释义、音标
   - 修改后点击保存

3. **删除单词**
   - 点击单词旁边的删除图标（红色垃圾桶）
   - 确认后单词被删除

4. **在章节中添加单词**
   - 点击章节中的 **"添加单词"** 按钮
   - 输入新单词（用逗号或换行分隔）
   - 点击"添加"按钮

5. **添加新章节**
   - 滚动到页面底部"添加新章节"区域
   - 输入单词列表
   - 点击"添加章节"按钮

6. **删除章节**
   - 点击章节右上角的删除图标
   - 确认后整个章节被删除

7. **更新单词信息**
   - 点击"更新单词信息"按钮
   - 系统会重新从词典中查找所有单词的释义和音标
   - 适用于添加新单词后批量获取释义

8. **删除整个词典**
   - 点击"删除词典"按钮
   - 确认后词典被永久删除

---

## 三、使用自定义词典

1. 在词典选择页面的"我的自定义词典"区域
2. 点击词典卡片即可开始练习
3. 练习模式与内置词典完全相同

## 四、词典优先级

系统按以下优先级查找单词信息（针对3年级儿童优化）：

1. 人教版三年级上下册
2. 新起点三年级
3. 外研新起点三年级
4. 冀教三年级
5. 人教版四、五、六年级
6. 外研小学版
7. 译林版高中
8. 初中、高中英语
9. 新概念英语
10. CET-4、常用基础词汇
11. 其他常用词库

## 五、数据存储与备份

### 1. 本地存储

- **浏览器端**：数据保存在 localStorage 中
- **Tauri 桌面端**：数据保存在本地文件系统

### 2. 导入/导出功能

在词典选择页面的"我的自定义词典"区域，有两个按钮：

- **导出**：将所有自定义词典导出为 Excel 文件（.xlsx）
- **导入**：从 Excel 文件导入词典

#### Excel 表格格式

| 列名 | 说明 |
|------|------|
| 单词 | 英文单词 |
| 释义 | 中文释义（多个用分号分隔） |
| 美式音标 | 美式发音 |
| 英式音标 | 英式发音 |
| 音标 | 其他音标 |
| 释义详情 | 详细解释 |
| 例句 | 例句 |
| 章节 | 所属章节 |

#### 导入/导出规则

- **同名称词典**：导入时会覆盖已有词典（保留原有 ID）
- **Sheet 名称**：每个 Sheet 对应一个词典，Sheet 名称即为词典名称
- **章节**：通过"章节"列自动分组

### 3. 服务器同步（可选）

如需多设备/多人共享词典，可配置后端服务器：

#### 启动后端服务

```bash
cd server
npm install
npm start
```

服务默认在 `http://localhost:3001` 启动。

#### 配置前端

编辑 `src/utils/customDictApi.ts`，设置服务器地址：

```typescript
export const API_BASE_URL = 'http://你的服务器IP:3001'
```

#### NAS Docker 部署

```bash
# 创建目录
mkdir -p /volume1/docker/qwerty-learner-server

# 运行容器
docker run -d \
  --name qwerty-learner-server \
  -p 3001:3001 \
  -v /volume1/docker/qwerty-learner-server/data:/app/data \
  node:18-alpine \
  sh -c "npm install && node server.js"
```

## 常见问题

**Q: 为什么有些单词找不到释义？**
A: 系统只在现有词典中查找，如果单词不在任何内置词典中，会显示"未找到释义"。您可以手动编辑添加释义。

**Q: 一个章节可以有多少个单词？**
A: 没有限制，您可以根据需要添加任意数量的单词。

**Q: 可以创建多少个自定义词典？**
A: 没有限制。

**Q: 如何分享我的词典？**
A: 有三种方式：
1. 导出为 Excel 文件，发送给其他人导入
2. 配置服务器同步，多人访问同一服务器
3. 导出后其他人通过导入功能加载
